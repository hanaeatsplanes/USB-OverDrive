## What even is this??

This is a USB-C -> 3x USB-A hub!

This uses:
* TUSB4041l as the main Hub
* 3x LP5912 as an LDO
* A male USB-C plug (JAE DX07P024AJ1) no need to worry about cables

## Why does this exist??

**This project was funded by [Hack Club Outpost](https://outpost.hackclub.com)!**

As a MacBook user, I deseperately need a dongle. I decided to design my own Hack Club themed dongle!
